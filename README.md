# CTF-MCP

A Model Context Protocol (MCP) server for CTF (Capture The Flag) challenges.

## Features

- **Crypto**: Cryptography tools for encoding/decoding, hash cracking, etc.
- **Web**: Web exploitation utilities
- **Pwn**: Binary exploitation helpers
- **Reverse**: Reverse engineering tools
- **Forensics**: Digital forensics utilities
- **Misc**: Miscellaneous CTF tools

## Installation

```bash
pip install -e .
```

## Usage

```bash
ctf-mcp
```
